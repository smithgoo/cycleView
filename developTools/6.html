iOS中set和get
set方法是传递一个参数没有返回 get是通过参数传递返回另一个参数
set方法：
-(void)setName:(NSString *)name
{
　　if (_name != name)
　　{
　　　[_name release];
　　　[name retain];
_name = name;
}
}
凡是.h里面的带星号变量或者属性的都要在dealloc中release一下 [_name release]；
这样释放 或者self.xxx = nil;这种更好
get方法：
-(UILabel *)titleLable
{
　　　 if (_titleLable == nil)
　　　 {
_titleLable = [[UILabelalloc]init];
} return_titleLable;
}
